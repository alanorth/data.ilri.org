Data resources at ILRI

Information and access to ILRI's data, tools, and code.

Data Portal

The Data Portal makes ILRI's open access datasets globally available. The platform is also the main access point for data from the CGIAR Research Program on Livestock and Fish (CRP) and from other CGIAR research programs and centres as part of our cross-centre collaboration.

Learn more »

Tools

The tools portal provides metadata and documentation for a wide range of research tools used by the CGIAR and our partners. It also enables interaction among users and offers a platform to feedback on adaptation and use of the tools.

Learn more »

Code

Source code for a handful of ILRI projects is hosted on GitHub, promoting an open-by-default workflow and enabling community interaction. The code includes systems written in house as well as community-maintained open-source projects we use.
